# pose-estimation
Tests/demos of various pre-trained pose estimation models.

[Example Video](https://youtu.be/N1KlFnjoEtg)

## Docker Environment
Building image:
```
docker-compose build
```

Running environment:
```
docker-compose run --rm app
```
